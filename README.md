# QWeather


利用和风天气的api（建议用专业开发者的，否则无法获取逐时天气）获取天气信息+一言+推送到自己的tg频道

根据wafu.py的可获取信息，选择内容生成天气信息，然后推送到tg频道就行

具体看wafu.py的注释和和风天气的开发文档

如果vps运行报错，建议把emoji去掉换普通字符

需要安装 sxtwl 和 python-telegram-bot 其他模块都是python自带的
